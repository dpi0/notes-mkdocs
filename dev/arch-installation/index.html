    arch linux install guide

    Arch Installation

    the holy grail: https://wiki.archlinux.org/title/installation_guide

    download ISO and verify¶

    always download the latest iso: https://archlinux.org/download/

    and the PGP signature: https://archlinux.org/download/#checksums

    ensure both the .iso and .sig are in the same directory then

    sudo pacman-key -v archlinux-version-x86_64.iso.sig

    optional - verify UEFI boot¶

    cat /sys/firmware/efi/fw_platform_size

    setup font and keymaps¶

    setfont ter-132n

loadkeys us

    setup internet¶

    iwctl -P "password" station wlan0 connect "SSID"
ping archlinux.org

    update system clock¶

    timedatectl set-timezone Asia/Kolkata
timedatectl set-ntp true
timedatectl

    create, format and mount partitions¶

    lsblk -f
cfdisk /dev/sdX

    important to set the partition type and file system as mentioned exactly below

    https://wiki.archlinux.org/title/Partitioning

    • /efi EFI System fat32 = 200M (contains just a bunch of .efi files, on dual boot NO NEED TO CREATE THIS AGAIN, share it with the already present 200M sda1 created using windows)
    • /boot Linux extended boot ext4 = 1G (contains the bootloader, the Kernel and initramfs images, useful to have this as a seperate partition for security and later customizations)
    • / Linux root (x86-64) ext4 = 35G (swapfile later in this)
    • /home Linux home ext4 = 20G
    • /data /shared etc Linux filesystem ext4
    mkfs.fat -F32 /dev/sdX-efi

mkfs.ext4 /dev/sdX-boot
mkfs.ext4 /dev/sdX-root
mkfs.ext4 /dev/sdX-home
mkfs.ext4 /dev/sdX-data

    always mount /mnt first!

    mount /dev/sdX-root /mnt

    mkdir /mnt/efi /mnt/boot /mnt/home /mnt/shared

    mount /dev/sdX-efi-or-shared-efi /mnt/efi

mount /dev/sdX-boot /mnt/boot
mount /dev/sdX-home /mnt/home
mount /dev/sdX-data /mnt/data

    optional - configure mirrorlist¶

    cp /etc/pacman.d/mirrorlist /etc/pacman.d/mirrorlist.bak
reflector --verbose -c India -l 20 -p https --sort rate --save /etc/pacman.d/mirrorlist

    install system and kernel¶

    pacstrap /mnt base linux linux-firmware vim sudo less intel-ucode

    generate the file system table¶

    genfstab -U /mnt >> /mnt/etc/fstab

    enter the installed ARCH system¶

    arch-chroot /mnt

    set up timezone and sync hardware clock¶

    ln -sf /usr/share/zoneinfo/Asia/Kolkata /etc/localtime

hwclock -w

    setup locales¶

    echo en_US.UTF-8 UTF-8 >> /etc/locale.gen

locale-gen

echo LANG=en_US.UTF-8 >> /etc/locale.conf
echo KEYMAP=us >> /etc/vconsole.conf

    set hostname¶

    echo arch >> /etc/hostname

    create user account¶

    useradd -mG wheel dpi0
passwd dpi0
echo "%wheel ALL=(ALL:ALL) ALL" >> /etc/sudoers

    install packages¶

    vim /etc/pacman.conf
---
---

pacman -S git openssh networkmanager base-devel pacman-contrib

systemctl enable fstrim.timer paccache.timer NetworkManager

    systemd-boot¶

    https://wiki.archlinux.org/title/systemd-boot#Installing_the_UEFI_boot_manager

    bootctl --esp-path=/efi --boot-path=/boot install

vim /efi/loader/loader.conf
---
default arch.conf
timeout 2
console-mode max
editor no
---

vim /boot/loader/entries/arch.conf
---
title Arch Linux
linux /vmlinuz-linux-lts
initrd /initramfs-linux-lts.img
options root=UUID=5fe2bd2b-3518-4fc3-8e5b-1277a1e97eae rw
---

    exit
umount -R /mnt

reboot

    post install - setup internet¶

    nmcli d wifi list
nmcli d wifi connect "SSID" password "lion9ns\!123"
nmcli c show

OR

nmtui

    disable root user¶

    sudo passwd -l root


sudo su -

    add hosts¶

    sudo tee -a /etc/hosts << EOF
192.168.1.100 server
EOF

    setup desktop environment and display manager¶

    sudo pacman -S hyprland firefox alacritty zsh kitty github-cli ly
sudo systemctl enable ly
reboot

    KDE:

    sudo pacman -S plasma-meta sddm
sudo systemctl disable ly
sudo systemctl enable sddm
reboot

    GNOME:

    sudo pacman -S gnome gnome-tweaks gnome-shell-extensions gdm mutter-dynamic-buffering
sudo systemctl disable ly
sudo systemctl enable gdm
reboot

    if using GRUB

    setup GRUB during installation

    first install: grub os-prober (os-prober only for multi boot detection)

    echo "GRUB_DISABLE_OS_PROBER=false" >> /etc/default/grub

grub-install --target=x86_64-efi --efi-directory=/efi --bootloader-id=GRUB

mkdir -p /efi/EFI/Boot
cp /efi/EFI/GRUB/grubx64.efi /efi/EFI/Boot/bootx64.efi

grub-mkconfig -o /boot/grub/grub.cfg

    to detect windows from GRUB after installation

    Note

    Note: You won't see Windows just yet

    To detect Windows, you need to use os-prober

    Important: This can only be done reliably outside of chroot

    sudo os-prober
sudo grub-mkconfig -o /boot/grub/grub.cfg

reboot

    Caution

    os-prober is inherently insecure as it mounts all partitions
    on your disk using grub-mount to check them for other OS,
    which is not a nice thing to do as root as you can exploit
    bugs in the filesystem code easily.

    ONLY SOLUTION TO THIS IS TO USE A DIFFERENT BOOTLOADER like rEFInd or systemd-boot and leave GRUB for good
