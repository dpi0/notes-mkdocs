    if - error: failed retrieving file on a package¶

    [!CAUTION] don't go for the immediate sudo pacman -Syu though it updates the package database as well, but here the issue is with the mirrors

    first try

    sudo cp /etc/pacman.d/mirrorlist /etc/pacman.d/mirrorlist.bak
sudo reflector --verbose -c India -l 20 -p https --sort rate --save /etc/pacman.d/mirrorlist

    if that doesn't solve, only then sudo pacman -Syu

    duplicate file finder, remover or mover¶

    for me pkolaczk/fclones works really well (even though development has essentially stopped)

    export DIRECTORY=/hdd/1TB-HDD-Backup/photos-backup/div
fclones group --output $DIRECTORY.json --format json --hidden $DIRECTORY

mkdir -p $DIRECTORY/redundant
fclones move $DIRECTORY/redundant <$DIRECTORY.json

fclones remove <$DIRECTORY.json

    another great tool with a focus on the GUI is qarmin/czkawka

    czkawka dup -d /hdd/Media/Photos/Div -p /hdd/Media/Photos/results.json -D aen

czkawka dup --search-method name -p ~/photos.json -d /hdd/Media/Photos/Div/Photos -D aen

    Methods to delete the files:

    • AEN - All files except the newest,
    • AEO - All files except the oldest,
    • ON - Only 1 file, the newest,
    • OO - Only 1 file, the oldest
    • AEB - All files except the biggest,
    • AES - All files except the smallest,
    • OB - Only 1 file, the biggest,
    • OS - Only 1 file, the smallest
    • HARD - create hard link
    • NONE - not delete files

    fix brave on wayland

    in about:flags, for Scroll left and right for History Navigation

    • Preferred Ozone platform = Wayland
    • For this you have to start Chromium with --enable-features=TouchpadOverscrollHistoryNavigation.
    • To persist this, copy /usr/share/applications/chromium-browser.desktop to ~/.local/share/applications
    • If you use Chrome or Chromium-Freeworld replace chromium-browser.desktop with chromium-freeworld.desktop or google-chrome.desktop
    • Edit the copied file, jump to the line that begins with Exec= and append --enable-features=TouchpadOverscrollHistoryNavigation after %U.
