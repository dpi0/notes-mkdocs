    record a macro:

    • Press qa to start recording into register a.
    • Execute your commands (viwsa*).
    • Press q to stop recording.

    Replaying the Macro:

    • Press @a to run the macro once.
    • Press 5@a to run the macro 5 times.

    1.4 Registers¶

    registers as a bunch of spaces in memory that vim uses to store some text. Each of these spaces have a identifier, so it can be accessed later
    Every register is accessed using a double quote before its name. For example, we can access the content that is in the register r with "r.

    Registers in Neovim allow you to store and recall pieces of text. There are several types of registers:

    • Unnamed register ("): The default register used for yank, delete, and paste operations.
    • Named registers ("a to "z): You can explicitly specify registers to store text.
    • Numbered registers ("0 to "9): Automatically used registers for recent yanks and deletes.
    • System clipboard register ("+ and "*): Interact with the system clipboard.

    Using Registers

    • Yank into a specific register: "ayw yanks a word into register a.
    • Paste from a specific register: "ap pastes the content of register a.
